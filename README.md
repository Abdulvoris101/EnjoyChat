# EnjoyChat

`Enjoychat` - a social media created in the django framework for people to communicate with each other and have fun by posting

Demo - http://insta.aerkinov.pro/

# Features

 - Posts
 - Profiles
 - Chat
 - Verifacation profiles
 - Notifications
 - Like system
 - Sharing posts

# Used stacks
 - Django
 - Vue js cdn
 - Mysql
 - Bootsrap
 - Redis Cache
 - Django channels
